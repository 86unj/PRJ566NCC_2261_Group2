# PRJ566NCC_2261_Group2

## Project Name
Foundit – Lost & Found Digital Platform for Academic Institutions

## Project Summary
Foundit is a B2B SaaS digital lost and found platform for academic institutions that centralizes item listing and claim verification in a secure, admin-controlled system, improving recovery efficiency and reducing manual workload.
